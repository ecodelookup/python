# OPERATORS

Python divides the operators in the following groups:

- Arithmetic operators
- Assignment operators
- Comparison operators
- Logical operators
- Identity operators
- Membership operators
- Bitwise operators

##  Arithmetic Operators

### Addition 
~~~
x = 5
y = 3
print(x + y) => 8
~~~

### Subtraction
~~~
x = 5
y = 3
print(x - y) => 2
~~~
